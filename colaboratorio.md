# Colaboratorio
## ¿Que es Google Colab?
Colaboraty es un aplicación muy util para el alumno, ya que se dispone de muchas herramientas y no es necesario disponer de una aplicacion externa al navegador para poder ejecutar un codigo. Ademas, se puede ir agregando anotaciones o ayudas de memoria dentro del mismo codigo. Otra funcion muy util es que se puede trabajar en un mismo codigo entre varias personas a la vez desde computadoras separadas, como contra se podria poner que la duración maxima es de 4 horas por cuenta.

1-Crear el cuaderno:
Primero entrás a [https://colab.research.google.com/?hl=es](https://colab.research.google.com/?hl=es) y ya te aparece la opción para crear un cuaderno nuevo. Cuando lo tengas, lo configurás desde *Editar > Configuración del cuaderno*. Ahí podés elegir el lenguaje, si querés usar GPU o TPU, y el entorno de ejecución.

2- Agregar texto:
Arriba de todo vas a ver el botón **“+ Texto”**. Si hacés click ahí, podés empezar a escribir. Colab usa Markdown, que es bastante simple. Por ejemplo, con # hacés títulos más grandes. Para poner algo en negrita lo rodeás con ** y para cursiva con *. También podés usar * para hacer listas.

3- Agregar código:
Al lado del botón de texto está **“+ Código”**. Desde ahí podés escribir tu propio código o incluso ayudarte con inteligencia artificial para generarlo.
